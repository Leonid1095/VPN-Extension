Установка расширения

PLGames Connect — менеджер сетевых профилей для браузера. Получайте готовые профили PLGames Pro в один клик.

Скачать расширение
  1. Распакуй архив в любую папку.
  2. В Chrome / Edge / Brave открой chrome://extensions/.
  3. Включи «Режим разработчика» в правом верхнем углу.
  4. Нажми «Загрузить распакованное» и выбери папку с расширением.
  5. Открой попап и нажми «PLGames Pro» для покупки подписки.
Что внутри:
  • Покупка подписки прямо из попапа через DonatePay
  • Профиль активируется автоматически после оплаты
  • Переключение профилей одним кликом
